# nvim
All my config for neovim

## Remaps

| Old key | Remap | Use |
|---------|-------|-----|
| \<leader\> | \<spacebar\> | Spacebar has now the leader use for instructions |
| :Ex | \<leader\>pv | Back to explorer |

## Cheatsheet

| Key | Use |
|-----|-----|
|  %  | Create file |
|  d  | Create directory |
| :w  | Write changes |
| :q  | Quite editor |
| :Ex | Explorer |
